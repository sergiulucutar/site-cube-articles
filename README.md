# SL Boilerplate

A base for developing websites

## Getting Started

### Dependencies

* [nodejs](https://nodejs.org/en/)
* [prismic (cms)](https://prismic.io/)

### Installing

```
npm install
```

### Executing program

* Start Express server
```
npm run be:build
```

* Start FE development server
```
npm run fe:development
```

* Build FE sources
```
npm run fe:build
```
